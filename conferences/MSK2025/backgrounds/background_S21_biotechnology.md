## 🎯 세션 개요

### 왜 이 세션이 중요한가?

## 🌟 쉬운말 풀이 (세션 전체)

### 이 세션은 쉽게 말하면...

**미생물 생명공학**: 미생물을 활용해서 실생활에 유용한 물질을 만드는 기술.

**4가지 연구 분야**:
1. **붉은 색소 (bacterioruberin)**: 항산화 효과, 화장품/건강식품용
2. **미세조류**: 바이오연료, 고부가 화학물질 생산
3. **포스트바이오틱스**: 유산균이 만드는 건강 물질
4. **녹조 제거**: 호수를 뒤덮는 녹조 해결

### 🎬 이야기로 이해하기

**[문제들]**
1. 화석연료 고갈 → 대체 에너지 필요
2. 항산화제 수요 증가 → 천연 색소 원함
3. 프로바이오틱스 한계 → 더 안정적인 대안 필요
4. 녹조 대발생 → 수질 오염, 독소 생성

**[미생물 해결사들]**

**1. 붉은 고세균 (S21-1)**
극한환경 (염전, 사해)에 사는 고세균이 **bacterioruberin**이라는 붉은 색소를 만듦.
- 비타민 A의 50배 항산화 효과!
- 문제: 고세균은 키우기 어려움
- 해결: 유전자를 대장균으로 이동 → 대량생산

**2. 미세조류 공장 (S21-2)**
미세조류는 "바다의 식물":
- 광합성으로 CO₂ 흡수
- 오일 생산 (바이오디젤 원료)
- 고부가 물질 (아스타잔틴, EPA) 생산
- 통합 접근법: 유전자 편집 + 배양 최적화

**3. 포스트바이오틱스 (S21-3)**
프로바이오틱스 (유산균) → 포스트바이오틱스 (유산균이 만든 물질)
- 장점: 살아있는 균 필요 없음, 안정적
- *Lactobacillus gasseri* 연구: 탄소원에 따라 다른 물질 생산
- 맞춤형 건강기능식품 개발

**4. 녹조 잡는 광물 (S21-4)**
녹조 (시아노박테리아) 문제:
- 호수 전체를 녹색으로 덮음
- 마이크로시스틴 (독소) 생성
- 전통 방법: 화학약품 (환경 오염)
- 새 방법: 광물 기반 물질로 선택적 제거

**[의미]**
이 연구들은 모두 **실용화 가능**한 기술:
- 이미 일부는 상용화 단계 (미세조류 바이오디젤)
- 스타트업 창업 아이템
- 환경 문제 + 경제적 가치 동시 해결

### 💡 왜 중요한가요?

**경제적**: 바이오산업 = 한국 미래 먹거리
**환경적**: 녹조 해결, CO₂ 감축
**건강**: 새로운 건강기능식품, 화장품 원료

## 📋 발표별 상세 분석

### 🔹 발표 1 (14:00-14:30): Rare C50 Carotenoid Production

**연사**: Myung-Ji Seo (Incheon National University)
**제목**: Engineering rare C50 carotenoid bacterioruberin production in *Corynebacterium glutamicum*

#### 🌟 쉬운말 풀이 (S21-1)

**이 발표는 쉽게 말하면**: 극한환경 미생물의 붉은 색소를 대량생산하는 기술

**[발견]**: 사해나 염전 같은 극한환경에 사는 **고세균(Haloarchaea)**은 **bacterioruberin**이라는 강력한 붉은 색소를 만듭니다. 이 색소는 비타민 A보다 **50배 강한** 항산화 효과를 가지고 있어 화장품과 건강식품 업계가 주목하고 있습니다.

**[문제와 해결]**: 하지만 고세균은 배양이 까다롭고 느립니다. 서명지 교수팀은 이 고세균의 색소 합성 유전자를 산업용 미생물인 **Corynebacterium glutamicum**(아미노산 생산 균)으로 이식했습니다. 마치 "Ferrari 엔진을 Toyota에 장착"하듯, 빠르게 자라는 균에 고부가 색소 생산 능력을 부여한 것입니다.

**[의미]**: 성공하면 천연 붉은 색소를 대량생산하여 합성 색소를 대체할 수 있습니다. 피부 노화 방지 화장품, 건강기능식품 등에 활용 가능합니다.

#### 최근 연구 배경

**Carotenoids의 중요성**

카로티노이드는 자연계에 널리 분포하는 색소로 다양한 기능을 가집니다:

1. **카로티노이드 분류**
   - **C40 carotenoids**: β-carotene (당근), lycopene (토마토), astaxanthin (연어)
     - 대부분의 자연 카로티노이드
     - 산업적으로 대량생산 (미생물, 화학합성)
   - **C50 carotenoids**: bacterioruberin, C.p. 450 등
     - 매우 희귀, 주로 고세균(archaea)에서 생산
     - 독특한 구조와 뛰어난 항산화 활성

2. **Bacterioruberin의 특징**
   - 화학 구조: C50 carotenoid, 4개의 하이드록시기
   - 색상: 진한 붉은색 (붉은 염전 색깔의 원인)
   - 항산화 활성: β-carotene 대비 **50배** (in vitro)
   - 막 안정화: 극한 환경 적응 (고염, 자외선)

3. **응용 분야**
   - 화장품: 항노화, 자외선 차단
   - 건강기능식품: 항산화, 항염증
   - 의약품: 신경보호, 심혈관 보호
   - 시장 가치: 매우 높음 (희소성)

**Haloarchaea의 bacterioruberin 생합성**

고세균은 극한 염도 환경에서 bacterioruberin을 생산합니다:

1. **생합성 경로**
   ```
   IPP/DMAPP (C5)
   ↓ (이소프레노이드 경로)
   GGPP (C20)
   ↓ (CrtB: phytoene synthase)
   Phytoene (C40)
   ↓ (탈포화 효소)
   Lycopene (C40)
   ↓ (CrtYe: lycopene elongase)
   Dihydrobisanhydrobacterioruberin (C50)
   ↓ (수화효소)
   Bacterioruberin (C50)
   ```

2. **핵심 효소**
   - **CrtYe (lycopene elongase)**: C40 → C50 전환 (핵심!)
   - **CrtB (phytoene synthase)**: C20 → C40
   - 탈포화효소, 수화효소 등

3. **대표적 생산균**
   - *Halobacterium salinarum*
   - *Haloferax mediterranei*
   - 생장 조건: 고염 배지 (15-25% NaCl), 느린 생장

**이종숙주에서 C50 생산의 도전**

Bacterioruberin을 대량생산하려면 산업용 미생물로 이식 필요:

1. **기존 시도와 한계**
   - *E. coli*에서 시도: 일부 성공, 하지만 생산량 낮음
   - 효소 활성, 전구체 공급, 대사 불균형 문제

2. ***C. glutamicum*의 장점**
   - 아미노산 (글루탐산, 라이신) 산업 생산균
   - GRAS (Generally Recognized As Safe) 인증
   - 강력한 대사 플럭스, 유전자 도구 확립
   - IPP/DMAPP 전구체 풍부 (이소프레노이드 경로)

3. **대사공학 전략**
   - 이소프레노이드 경로 강화 (MEP pathway)
   - Haloarchaea 유전자 이식 및 발현 최적화
   - 경쟁 경로 차단
   - 배양 조건 최적화

#### 예상 발표 내용

1. **Bacterioruberin 생합성 유전자 클러스터**
   - *Haloferax* 또는 *Halobacterium*에서 유전자 발굴
   - 핵심 유전자: *crtB*, *crtI*, *crtYe*, 수화효소
   - 유전자 클러스터 구성

2. ***C. glutamicum* 대사공학**
   - MEP pathway 강화 (IPP/DMAPP 공급 증대)
     - *dxs*, *idi* 과발현
   - Carotenoid 생합성 유전자 도입
     - 코돈 최적화, 프로모터 선택
   - 경쟁 경로 차단
     - *crtE* (endogenous) 조절

3. **발현 최적화**
   - 오페론 구성 최적화
   - RBS (ribosome binding site) 강도 조절
   - 효소 발현 밸런싱

4. **Bacterioruberin 생산 및 분석**
   - 생산량 (mg/L 또는 mg/g DCW)
   - HPLC, LC-MS로 구조 확인
   - 색상, 흡광도 스펙트럼

5. **배양 최적화**
   - 탄소원, 질소원 최적화
   - 산소 공급, 온도, pH
   - Fed-batch 또는 연속 배양

6. **응용 가능성**
   - 항산화 활성 측정 (DPPH, ABTS assay)
   - 화장품/건강식품 제형화
   - 경제성 분석

**연사**: Yoon-E Choi (Korea University)

#### 🌟 쉬운말 풀이 (S21-2)

**이 발표는 쉽게 말하면**: 미세조류를 "바다의 공장"으로 만들기

**[배경]**: **미세조류**(Microalgae)는 현미경으로 보이는 작은 조류로, 바다와 호수에 살며 광합성을 합니다. 식물보다 **10-50배** 빠르게 자라고, 오일(바이오디젤 원료)과 고부가 화학물질(아스타잔틴, EPA)을 생산합니다.

**[연구 접근]**: 최윤이 교수팀은 **통합 접근법**을 사용합니다:
1. **유전자 편집**: CRISPR로 오일 생산 유전자를 강화
2. **배양 최적화**: 빛 조건, 영양분, 온도를 조절해 생산성 극대화
3. **광합성 + 발효 혼합**: 낮에는 빛으로, 밤에는 유기물로 자라게 함

**[의미]**: 미세조류는 탄소중립 바이오연료의 유망 후보입니다. 경작지가 필요 없고(바다에서 재배), CO₂를 흡수하며, 식용유보다 훨씬 많은 오일을 생산합니다. 또한 연어 사료(EPA), 건강식품(스피룰리나) 등으로 활용됩니다.

#### 최근 연구 배경

**미세조류의 잠재력**

미세조류는 지속가능한 바이오 경제의 핵심 자원입니다:

1. **미세조류의 장점**
   - **빠른 생장**: 세포 분열 시간 수 시간~1일 (육상식물 대비 10-50배)
   - **높은 생산성**: 단위 면적당 바이오매스 생산량이 육상작물보다 훨씬 높음
   - **CO₂ 흡수**: 광합성으로 CO₂ 고정 (온실가스 감축)
   - **경작지 불필요**: 바다, 사막, 폐수 등 활용
   - **다양한 생산물**: 오일, 단백질, 카로티노이드, 오메가-3, 바이오플라스틱 전구체

2. **주요 미세조류 종**
   - ***Chlorella*** (클로렐라): 고단백, 건강식품
   - ***Spirulina*** (스피룰리나): 단백질 60-70%, 슈퍼푸드
   - ***Haematococcus pluvialis***: 아스타잔틴 생산 (항산화제)
   - ***Nannochloropsis***: EPA (오메가-3) 생산
   - ***Dunaliella salina***: β-carotene 생산
   - ***Chlamydomonas reinhardtii***: 모델 생물, 유전자 도구 발달

3. **응용 분야**
   - **바이오연료**: 바이오디젤, 바이오에탄올, 바이오가스
   - **식품**: 건강기능식품, 단백질 보충제
   - **사료**: 양식 어류 사료 (EPA, DHA)
   - **화장품**: 항산화제, 보습제
   - **의약품**: 항염증, 항암 화합물
   - **환경**: CO₂포집, 폐수 처리

**미세조류 바이오연료의 도전과 기회**

1. **왜 바이오연료인가?**
   - 화석연료 고갈, 온실가스 문제
   - 1세대 바이오연료 (옥수수, 대두)의 한계: 식량 경쟁
   - 2세대 (리그노셀룰로스)의 한계: 전처리 비용
   - 3세대 바이오연료: 미세조류 (식량 경쟁 없음)

2. **미세조류 오일의 장점**
   - 오일 함량: 건조중량의 20-50% (일부 균주 70% 이상)
   - 대두 (20%), 팜유 (30%)보다 높음
   - 단위 면적당 오일 생산량: 팜유 대비 5-10배

3. **상업화의 장벽**
   - **생산 비용**: 여전히 화석연료보다 비쌈
   - **배양 규모**: 대규모 배양 시스템 (open pond vs photobioreactor)
   - **수확 비용**: 작은 세포 크기, 낮은 밀도
   - **오일 추출**: 에너지 소비
   - **생산성**: 더 높은 오일 함량 필요

**통합 접근법의 필요성**

단일 전략으로는 상업화 어려움 → **유전공학 + 배양 최적화**를 통합해야 경제성 확보:

1. **유전공학 (Genetic Engineering)**
   - 오일 함량 증대: 지질 생합성 경로 강화
   - 생장 속도 향상: 광합성 효율 개선
   - 스트레스 내성: 고온, 고염, 오염 내성
   - 고부가 화합물 생산

2. **배양 최적화 (Cultivation Optimization)**
   - 빛 조건: 광도, 광주기, 파장
   - 영양분: 질소, 인, 미량원소
   - CO₂ 공급, pH, 온도
   - 배양 방식: photoautotrophic vs heterotrophic vs mixotrophic

3. **시스템 통합**
   - 유전자 편집으로 개선된 균주 + 최적 배양 조건
   - 전과정 평가 (LCA), 기술경제성 분석 (TEA)

#### 예상 발표 내용

1. **통합 접근법 개요**
   - 유전공학과 배양 최적화의 시너지
   - 타겟 미세조류 종 (*Chlorella*, *Chlamydomonas*, *Nannochloropsis* 등)

2. **유전공학 전략**
   - **CRISPR/Cas9 유전자 편집**
     - 표적 유전자: 지질 생합성 (*ACC*, *DGAT*), 전분 합성 억제
     - 녹다운 (knockdown) 또는 녹아웃 (knockout)
   - **지질 생합성 경로 강화**
     - Acetyl-CoA carboxylase (ACC) 과발현
     - Diacylglycerol acyltransferase (DGAT) 과발현
   - **경쟁 경로 차단**
     - 전분 합성 억제 → 탄소 플럭스를 오일로 유도
   - **광합성 효율 개선**
     - 안테나 색소 조절 (과도한 빛 흡수 방지)
     - CO₂ 농축 기작 (CCM) 강화

3. **배양 최적화**
   - **광독립영양 (Photoautotrophic) 배양**
     - 빛 + CO₂로 생장
     - Open pond vs Photobioreactor (PBR)
     - 빛 투과, 혼합 최적화
   - **종속영양 (Heterotrophic) 배양**
     - 유기물 (포도당, 글리세롤)로 생장
     - 빛 불필요, 고밀도 배양 가능
     - 바이오리액터 활용
   - **혼합영양 (Mixotrophic) 배양**
     - 빛 + 유기물 동시 이용
     - 생산성 극대화
   - **영양분 조건**
     - 질소 제한 → 오일 축적 유도
     - CO₂ 농도, pH 조절

4. **고부가 화합물 생산**
   - **아스타잔틴** (*Haematococcus*)
     - 스트레스 조건 (강광, 질소 결핍)으로 유도
   - **EPA/DHA** (*Nannochloropsis*, *Schizochytrium*)
   - **β-carotene** (*Dunaliella*)

5. **수확 및 가공**
   - 응집, 원심분리, 여과
   - 오일 추출 (용매, 초임계 CO₂)
   - 바이오디젤 전환 (transesterification)

6. **경제성 및 환경 영향**
   - 기술경제성 분석 (TEA)
   - 전과정 평가 (LCA): CO₂ 감축 효과

### 🔹 발표 3 (15:00-15:30): Lactobacillus Postbiotics

**연사**: Eun Ju Yun (Catholic University of Korea)
**제목**: Carbon source-dependent metabolic responses of *Lactobacillus gasseri* and implications for postbiotic development

#### 🌟 쉬운말 풀이 (S21-3)

**이 발표는 쉽게 말하면**: 유산균이 만드는 건강 물질 (포스트바이오틱스) 맞춤 생산

**[개념 변화]**:
- **프로바이오틱스 1.0**: 살아있는 유산균을 먹기 (요구르트, 김치)
- **포스트바이오틱스 2.0**: 유산균이 만든 물질만 추출 (더 안정적, 보관 쉬움)

**[연구 내용]**: 윤은주 교수팀은 ***Lactobacillus gasseri***(장 건강 유산균)에게 다양한 "먹이"(탄소원)를 주면서 어떤 대사산물을 만드는지 분석했습니다. 놀랍게도:
- **포도당**을 주면 → 젖산 많이 생산
- **프락토올리고당**을 주면 → 짧은사슬지방산(SCFA) 생산
- **식이섬유**를 주면 → 항염증 물질 생산

마치 "요리 재료에 따라 다른 요리가 나오듯", 먹이에 따라 다른 건강 물질이 나옵니다!

**[의미]**: 이제 우리는 원하는 건강기능식품을 "주문 제작"할 수 있습니다. 예: "변비 개선용 SCFA", "항염증용 대사산물" 등.

#### 최근 연구 배경

**프로바이오틱스에서 포스트바이오틱스로**

건강기능식품 시장의 패러다임이 변화하고 있습니다:

1. **프로바이오틱스 (Probiotics)**
   - 정의: 적절한 양을 섭취했을 때 건강에 이로운 살아있는 미생물
   - 예: *Lactobacillus*, *Bifidobacterium*
   - 장점: 장내 미생물총 개선, 면역 조절
   - 단점:
     - 생존성 문제 (위산, 담즙산에 약함)
     - 보관 조건 까다로움 (냉장)
     - 효과 개인차 (장내 환경에 따라)
     - 면역저하 환자에서 감염 위험

2. **프리바이오틱스 (Prebiotics)**
   - 정의: 프로바이오틱스의 먹이가 되는 물질 (난소화성 탄수화물)
   - 예: 이눌린, 프락토올리고당 (FOS), 갈락토올리고당 (GOS)
   - 장점: 안정적, 보관 쉬움
   - 단점: 효과가 간접적

3. **포스트바이오틱스 (Postbiotics)**
   - 정의: 프로바이오틱스가 생산하는 생리활성 대사산물
   - 예: 짧은사슬지방산 (SCFA), 세포벽 성분, 효소, 펩타이드
   - 장점:
     - 살아있는 균 불필요 → 안전성 높음
     - 상온 보관 가능 → 유통 편리
     - 정량화 가능 → 용량 조절 정확
     - 빠른 효과 → 직접 작용
   - 단점: 생산 비용

**포스트바이오틱스의 종류와 기능**

1. **짧은사슬지방산 (SCFA: Short-Chain Fatty Acids)**
   - 종류: 아세트산, 프로피온산, 뷰티르산
   - 생성: 탄수화물 발효
   - 기능:
     - 장 상피세포 에너지원 (특히 butyrate)
     - 장 장벽 기능 강화
     - 항염증 (T-reg 세포 유도)
     - 대사 조절 (인슐린 민감성 개선)

2. **세포벽 성분**
   - Lipoteichoic acid (LTA), peptidoglycan
   - 면역 조절, Toll-like receptor (TLR) 활성화

3. **세균소 (Bacteriocins)**
   - 항균 펩타이드
   - 병원균 억제

4. **효소**
   - β-galactosidase (유당 분해)
   - 담즙산 가수분해효소 (BSH)

5. **항산화 물질**
   - Glutathione, superoxide dismutase (SOD)

***Lactobacillus gasseri*의 특징**

*L. gasseri*는 인간 위장관에 자연적으로 존재하는 유산균:

미생물의 대사는 공급되는 영양분에 따라 달라집니다:

### 🔹 발표 4 (15:30-16:00): Cyanobacterial Bloom Control

**연사**: In-Hyun Nam (Korea Institute of Geoscience)
**제목**: Mineral-based materials for selective control of cyanobacterial blooms: Environmental safety assessment and field application strategies

#### 🌟 쉬운말 풀이 (S21-4)

**이 발표는 쉽게 말하면**: 녹조를 안전하게 제거하는 광물 기반 기술

**[문제]**: 여름마다 호수와 강이 녹색으로 뒤덮이는 **녹조 현상**. 원인은 **시아노박테리아**(남세균)의 대번식입니다. 문제는:
- 물이 썩는 냄새
- **마이크로시스틴** 독소 생성 (간 손상)
- 물고기 떼죽음
- 수돗물 오염 우려

**[기존 해결책의 한계]**: 황산구리 같은 화학약품을 뿌리면 녹조는 죽지만, 물고기와 다른 수생식물도 함께 죽습니다. "암세포 죽이려고 정상 세포도 죽이는" 격입니다.

**[새로운 해결책]**: 남인현 박사팀은 **광물 기반 물질**을 개발했습니다. 이 물질은:
- 시아노박테리아만 **선택적**으로 제거
- 물고기와 다른 플랑크톤은 안전
- 환경 친화적 (자연 분해)

원리: 광물 입자가 시아노박테리아 표면에만 특이적으로 붙어서 침전시킴.

**[의미]**: 대청호, 낙동강 같은 식수원의 녹조를 안전하게 제어할 수 있습니다. 여름철 물 부족 문제 해결에 기여합니다.

#### 최근 연구 배경

**시아노박테리아 번식 (Cyanobacterial Bloom) 문제**

녹조 현상은 전 세계적인 환경 문제입니다:

1. **시아노박테리아 (Cyanobacteria)**
   - 다른 이름: 남세균 (blue-green algae)
   - 특징: 광합성 세균, 질소 고정 능력
   - 대표 종: *Microcystis aeruginosa*, *Anabaena*, *Oscillatoria*

2. **녹조 발생 원인**
   - **부영양화 (Eutrophication)**
     - 과도한 영양염류 (질소, 인) 유입
     - 농업 비료, 축산 폐수, 생활하수
   - **기후 변화**
     - 수온 상승 → 시아노박테리아 생장 촉진
     - 강우 패턴 변화
   - **수리 구조물**
     - 댐, 보 → 물 체류 시간 증가

3. **녹조의 피해**
   - **독소 생성**
     - Microcystin (마이크로시스틴): 간독성
     - Cylindrospermopsin: 신장독성
     - Anatoxin: 신경독
     - 사람, 동물 건강 위협
   - **수질 악화**
     - 용존 산소 고갈 (야간 호흡, 사멸 후 분해)
     - 물고기 떼죽음
     - 악취 (geosmin, 2-MIB)
   - **경제적 손실**
     - 식수원 오염 → 정수 비용 증가
     - 관광, 양식업 피해

4. **한국의 녹조 현황**
   - 4대강 (한강, 낙동강, 금강, 영산강)에서 반복 발생
   - 여름철 (6-9월) 집중
   - 2013-2015년 극심한 녹조 발생
   - 정부 대응: 녹조 경보제, 조류 제거선

**기존 녹조 제거 기술과 한계**

1. **화학적 방법**
   - **황산구리 (CuSO₄)**
     - 시아노박테리아 세포 파괴
     - 문제: 물고기, 다른 조류도 피해, 구리 축적
   - **과산화수소 (H₂O₂)**
     - 산화 스트레스로 세포 사멸
     - 문제: 고농도 필요, 비선택적

2. **물리적 방법**
   - **수류 조절, 폭기**
     - 산소 공급, 혼합 촉진
     - 문제: 대규모 적용 어려움, 에너지 소비
   - **초음파**
     - 세포 파괴
     - 문제: 효과 제한적, 비용

3. **생물학적 방법**
   - **바이오 조작 (Biomanipulation)**
     - 포식자 (동물 플랑크톤) 증가
     - 문제: 생태계 교란 위험
   - **미생물 제제**
     - 시아노박테리아 억제 미생물
     - 문제: 효과 불안정

4. **영양염류 저감**
   - 인 제거 (응집제)
   - 문제: 근본적 해결 어려움, 이미 축적된 영양염

**광물 기반 제어 물질의 원리**

선택적으로 시아노박테리아를 제거하는 신기술:

1. **응집-침전 (Flocculation-Sedimentation)**
   - 광물 입자가 시아노박테리아에 흡착
   - 밀도 증가 → 침전
   - 수표면에서 제거

2. **선택성 메커니즘**
   - 시아노박테리아 표면 특성 (EPS, 전하) 이용
   - 광물 표면 개질로 선택성 향상
   - 다른 조류, 물고기 영향 최소화

3. **광물 종류**
   - **점토 (Clay)**: kaolin, montmorillonite
   - **변성 점토 (Modified clay)**: 양이온 계면활성제 처리
   - **철 기반 물질**: 철 산화물, 수산화철
   - **인 흡착제**: 란타늄 변성 점토

4. **장점**
   - 환경 친화적 (자연 물질)
   - 선택성 (타겟 시아노박테리아)
   - 비교적 저비용
   - 대규모 적용 가능

## 🧠 세션 전체 핵심 요약

### 미생물 생명공학의 4가지 실용 응용

이 세션은 **기초 연구 → 상업화**까지 연결되는 실용 기술을 제시합니다:

**1. 극한환경 → 고부가 물질 (S21-1)**
- Haloarchaea bacterioruberin → *C. glutamicum* 대사공학
- 항산화 색소 대량생산 → 화장품, 건강식품
- 메타게노믹스 → 신규 효소 발굴 응용

**2. 미세조류 → 탄소중립 바이오경제 (S21-2)**
- 유전공학 + 배양 최적화 통합
- 바이오디젤 + 고부가 화합물 동시 생산
- CRISPR 대사공학 → 오일 함량 증대

**3. 프로바이오틱스 → 포스트바이오틱스 (S21-3)**
- 살아있는 균 → 대사산물 (더 안전, 안정)
- 탄소원 조절 → 맞춤형 SCFA, 항균 물질
- 항생제 대체제 개발

**4. 녹조 제거 → 환경 친화 기술 (S21-4)**
- 광물 기반 선택적 제어
- 시아노박테리아만 제거, 생태계 보호
- 식수원 안전 확보

### 연구 간 시너지

```
극한환경 메타게놈 ─────┐
                      ├──→ 신규 효소/화합물 발굴
미세조류 메타게놈 ──────┘      │
                             ↓
                      대사공학 적용
                             │
                             ├──→ 고부가 물질 생산
                             │
                             └──→ 포스트바이오틱 생산

광물 나노기술 ──→ 항균 나노물질 개발
```

### 우리 연구와의 통합

**AMR 대응 전략:**
- 포스트바이오틱 항균 물질: 항생제 대체
- 대사공학: 항균 화합물 대량생산
- 선택적 제어: 병원균 억제, 유익균 보존

**메타게노믹스 응용:**
- 극한환경, 미세조류 메타게놈: 신규 유전자 발굴
- 생합성 경로 발굴 및 재구성
- 미탐사 항균 화합물 발견

**스타트업/산업화:**
- 천연 색소, 건강식품, 녹조 제거: 실용 기술
- 기술 이전, 상업화 전략 학습
