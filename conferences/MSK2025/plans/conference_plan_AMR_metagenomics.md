# MSK2025 Conference Plan - AMR & Metagenomics Focus

**연구자 프로필**: 박사후연구원
**연구 분야**: 메타지노믹스 기반 도심 미생물 AMR 감시·예측 & 항생제 개발
**참석 목적**: 최신 연구 동향, 네트워킹, 새로운 기법 학습

## 📊 시간대별 세션 선택 가이드

### 선택 기준 (우선순위)
1. **🎯 직접 연관성**: AMR, 항생제 개발, 진단 기법
2. **🧬 기법 관련성**: 메타지노믹스, 오믹스 분석 기법
3. **🌍 응용 분야**: 환경/도심 미생물, 감시 시스템
4. **🤝 네트워킹 가치**: 협업 가능성, 최신 기법 공유
