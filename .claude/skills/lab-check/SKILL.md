# Lab Check Skill v1.0
## 연구실/랩 검증 및 역량 분석

공동연구, 학생 진학, 포닥 지원, 기관간 협력 등을 위해 연구실(랩)의 전반적인 역량과 환경을 객관적으로 평가하는 skill입니다.

---

## 🎯 주요 기능

1. **PI(책임연구자) 배경 검증**
2. **연구 프로그램 분석** (주제, 방법론, 강점)
3. **랩 구성원 현황**
4. **연구 장비/인프라**
5. **졸업생 진로 추적**
6. **연구비/재정 현황**
7. **협업 네트워크**
8. **랩 문화/평판**

---

## 📋 워크플로우

### Step 1: 초기화 및 정보 수집

**사용자에게 질문:**
```
연구실(랩) 검증을 시작합니다.

1. 연구실명 또는 PI 이름:

2. 소속 기관/대학:

3. 연구 분야:

4. 관련 자료가 있나요? (선택)
   - 연구실 웹페이지 URL
   - PI CV 파일
   - 없음 (웹 검색만 진행)

5. 어떤 목적의 검증인가요?
   - 공동연구/협력
   - 대학원 진학 고려
   - 포닥 지원 고려
   - 기술이전/산학협력
   - 기관간 MOU
   - 기타
```

### Step 2: PI 배경 검증

**[researcher-check와 동일한 프로세스 적용]**

- 학력/경력 검증
- 논문 업적 (h-index, 인용수)
- 연구비 수주 이력
- 학회 리더십

**추가 PI 특성 분석:**

| 항목 | 내용 |
|------|------|
| 연구 스타일 | 이론/실험/계산/융합 |
| 지도 스타일 | Hands-on / Independent |
| 랩 운영 기간 | 설립 연도, 안정성 |
| 테뉴어 여부 | 직위 안정성 |

### Step 3: 연구 프로그램 분석

**조사 항목:**

| 항목 | 내용 |
|------|------|
| **핵심 연구주제** | 3-5개 메인 토픽 |
| **연구 방법론** | 실험/계산/이론/임상 등 |
| **기술적 강점** | 독보적 기술, 노하우 |
| **차별화 포인트** | 경쟁 랩 대비 |
| **연구 방향** | 현재 → 미래 계획 |

**최근 5년 연구 트렌드:**
- 주요 논문 키워드 변화
- 새로운 연구 방향
- 폐기된 연구 라인

### Step 4: 랩 구성원 현황

**WebSearch 쿼리:**
```
- "[랩명] members"
- "[PI명] lab team"
- "[랩명] people"
```

**구성원 매트릭스:**

| 직급 | 인원 | 세부 |
|------|------|------|
| 교수/PI | 1 | (공동 PI 있는 경우 추가) |
| 연구교수/Research Scientist | N | |
| 포닥 | N | |
| 박사과정 | N | 학년 분포 |
| 석사과정 | N | |
| 학부연구생 | N | |
| 연구원/테크니션 | N | |

**적정 규모 평가:**
- 총 인원 vs 연구비 규모
- 학생:포닥 비율
- PI 대비 학생 수 (지도 가능성)

### Step 5: 연구 장비/인프라

**조사 방법:**
- 랩 웹페이지 "Equipment" / "Facilities" 섹션
- 기관 공용장비 현황
- 최근 논문의 Methods 섹션

**장비 목록:**

| 장비명 | 용도 | 보유/공용 | 상태 |
|--------|------|-----------|------|
| ... | ... | 자체보유/공용 | 최신/구형 |

**인프라 평가:**

| 항목 | 상태 |
|------|------|
| 실험 공간 | 면적, 위치 |
| 계산 자원 | 클러스터, GPU |
| 생물자원 | 세포주, 동물시설 |
| 데이터베이스 | 자체 구축 DB |

### Step 6: 졸업생 진로 추적

**WebSearch 쿼리:**
```
- "[랩명] alumni"
- "[PI명] former students"
- "[PI명] PhD graduates"
```

**졸업생 진로 통계:**

| 진로 | 박사 | 석사 |
|------|------|------|
| 교수/연구원 | N% | - |
| 산업계 (대기업) | N% | N% |
| 산업계 (스타트업) | N% | N% |
| 박사 진학 | - | N% |
| 포닥 | N% | - |
| 기타 | N% | N% |

**주목할 졸업생:**
- 유명 교수가 된 졸업생
- 창업한 졸업생
- 주요 기업 임원

**졸업 소요 기간:**
- 평균 박사 과정 기간
- 조기 졸업 / 지연 졸업 비율

### Step 7: 연구비/재정 현황

**조사 항목:**

| 항목 | 내용 |
|------|------|
| 현재 진행 과제 | 제목, 기간, 금액 |
| 주요 기금원 | 정부/산업체/재단 |
| 연간 연구비 규모 | 추정액 |
| 재정 안정성 | 장기과제 여부 |

**산학협력 현황:**
- 기업 스폰서십
- 공동연구 계약
- 기술이전 실적

### Step 8: 협업 네트워크

**국내 협력:**

| 기관 | 분야 | 협력 형태 |
|------|------|-----------|
| ... | ... | 공동연구/장비공유/인력교류 |

**국제 협력:**

| 국가 | 기관 | PI | 협력 내용 |
|------|------|-----|-----------|
| ... | ... | ... | ... |

**산업계 파트너:**
- 기업 목록
- 협력 형태

### Step 9: 랩 문화/평판

**조사 방법:**
```
- "[랩명] review"
- "[PI명] mentor"
- Glassdoor / RateMyProfessors (해외)
- 대학원생 커뮤니티 (국내)
```

**평가 항목:**

| 항목 | 조사 내용 |
|------|-----------|
| 미팅 빈도 | 주간/개별 미팅 |
| 근무 시간 | 자율/의무적 |
| 휴가 정책 | 공식/비공식 |
| 졸업 지원 | 취업/진학 도움 |
| 랩 분위기 | 경쟁적/협력적 |

**Red Flags:**

| Flag | 심각도 |
|------|--------|
| 졸업생 부정적 후기 다수 | 🔴 높음 |
| 높은 중도 이탈률 | 🔴 높음 |
| 졸업 지연 패턴 | 🟡 중간 |
| PI 부재 잦음 | 🟡 중간 |
| 재정 불안정 신호 | 🟡 중간 |
