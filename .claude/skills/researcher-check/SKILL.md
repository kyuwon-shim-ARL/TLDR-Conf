# Researcher Check Skill v1.0
## 개인 연구자 검증 및 학술 프로필 분석

공동연구, 자문위원 위촉, 채용, 학술협력 등을 위해 개인 연구자의 학술적 역량과 배경을 객관적으로 검증하는 skill입니다.

---

## 🎯 주요 기능

1. **학력/경력 검증**
2. **논문 업적 분석** (인용수, h-index, 주요 논문)
3. **연구비 수주 이력**
4. **학회 활동/리더십**
5. **제자 배출 현황**
6. **연구 네트워크 분석**
7. **Red Flags 탐지**

---

## 📋 워크플로우

### Step 1: 초기화 및 정보 수집

**사용자에게 질문:**
```
개인 연구자 검증을 시작합니다.

1. 연구자 이름 (영문/한글):

2. 소속 기관 (알고 있다면):

3. 전공/연구분야:

4. 관련 자료가 있나요? (선택)
   - CV/이력서 파일 경로
   - 개인 웹페이지 URL
   - Google Scholar 프로필 URL
   - 없음 (웹 검색만 진행)

5. 어떤 목적의 검증인가요?
   - 공동연구 파트너
   - 자문위원/심사위원
   - 채용/초빙
   - 세미나 연사 초청
   - 학위논문 심사위원
   - 기타
```

### Step 2: 학력/경력 검증

**WebSearch 쿼리:**
```
- "[이름] [전공] PhD thesis"
- "[이름] curriculum vitae CV"
- "[이름] [대학명] professor"
- "[이름] LinkedIn profile"
- "[이름] ORCID"
```

**검증 항목:**

| 항목 | 검증방법 | 결과 |
|------|----------|------|
| 학사 | 대학 동문DB, LinkedIn | ✓/✗/? |
| 석사 | 학위논문 DB | ✓/✗/? |
| 박사 | ProQuest, 학위논문 DB | ✓/✗/? |
| 포닥 | 기관 웹사이트, 논문 소속 | ✓/✗/? |
| 현 소속 | 기관 faculty 페이지 | ✓/✗/? |

### Step 3: 논문 업적 분석

**정보원:**
- Google Scholar
- PubMed
- Web of Science
- Scopus
- ORCID

**수집 지표:**

| 지표 | 설명 | 값 |
|------|------|-----|
| **총 논문수** | 전체 발표 논문 | N편 |
| **1저자/교신저자** | 주요 기여 논문 | N편 |
| **총 인용수** | Google Scholar 기준 | N회 |
| **h-index** | h편의 논문이 각각 h회 이상 인용 | N |
| **i10-index** | 10회 이상 인용 논문 수 | N |
| **최근 5년 인용** | 연구 활성도 지표 | N회 |

**대표 논문 분석 (Top 5):**

| 순위 | 논문 제목 | 저널 | IF | 인용수 | 역할 |
|------|----------|------|-----|--------|------|
| 1 | ... | ... | ... | ... | 1저자/교신 |
| 2 | ... | ... | ... | ... | ... |

**연구 분야 키워드:**
- 주요 키워드 추출 (논문 제목/초록 기반)
- 연구 트렌드 변화

### Step 4: 연구비 수주 이력

**WebSearch 쿼리:**
```
- "[이름] NIH grant"
- "[이름] NSF funding"
- "[이름] 한국연구재단 과제"
- "[이름] research grant PI"
```

**수집 정보:**

| 과제명 | 기간 | 금액 | 역할 | 기금원 |
|--------|------|------|------|--------|
| ... | 20XX-20XX | $XXX | PI/Co-PI | NIH/NSF/NRF |

**평가:**
- 총 수주액 (PI 기준)
- 진행 중인 과제 수
- 대형과제 (센터급) 경험

### Step 5: 학회 활동/리더십

**조사 항목:**

| 활동 유형 | 내용 |
|-----------|------|
| **학회 임원** | 회장, 이사, 위원장 등 |
| **저널 에디터** | Editor-in-Chief, Associate Editor, Editorial Board |
| **학회 조직** | 학회 개최, 세션 오거나이저 |
| **기조/초청 강연** | Keynote, Plenary, Invited talks |
| **수상** | 학술상, 젊은과학자상 등 |

### Step 6: 제자 배출 현황

**조사 방법:**
```
- "[이름] lab alumni"
- "[이름] PhD students graduated"
- "[이름] mentorship"
- 연구실 웹페이지 → Alumni 섹션
```

**수집 정보:**

| 구분 | 인원 | 현재 진로 |
|------|------|-----------|
| 박사 졸업 | N명 | 교수 N, 산업계 N, 포닥 N |
| 석사 졸업 | N명 | 박사진학 N, 산업계 N |
| 포닥 배출 | N명 | 교수 N, 산업계 N |

**제자 성공 사례:**
- 유명 연구자로 성장한 제자
- 창업한 제자
- 수상한 제자

### Step 7: 연구 네트워크 분석

**공저자 네트워크:**
- 주요 공동연구자 (빈도순)
- 국제 협력 현황
- 산학협력 파트너

**소속 연구그룹:**
- 컨소시엄, 센터 참여
- 학회 분과 활동

### Step 8: Red Flags 탐지

| 카테고리 | Red Flag | 심각도 |
|----------|----------|--------|
| **학력** | 검증 안 되는 학위 | 🔴 높음 |
| **논문** | Predatory journal 다수 게재 | 🔴 높음 |
| **논문** | 자기인용 비율 과다 (>30%) | 🟡 중간 |
| **논문** | 최근 5년 논문 없음 | 🟡 중간 |
| **윤리** | Retraction 이력 | 🔴 높음 |
| **윤리** | 연구윤리 위반 기록 | 🔴 높음 |
| **경력** | 잦은 소속 변경 (< 2년) | 🟡 중간 |
| **펀딩** | PI 수주 이력 없음 (시니어인데) | 🟡 중간 |

**WebSearch 쿼리:**
```
- "[이름] retraction"
- "[이름] research misconduct"
- "[이름] plagiarism"
- "Retraction Watch [이름]"
```

### 연구 역량 등급

| 등급 | h-index | 총 인용 | 1저자/교신 | 연구비 |
|------|---------|---------|------------|--------|
| **S** (세계적) | 50+ | 10,000+ | 50+ | 대형과제 다수 |
| **A** (우수) | 30-49 | 3,000-10,000 | 30-49 | 중형과제 다수 |
| **B** (양호) | 15-29 | 1,000-3,000 | 15-29 | 소형과제 수주 |
| **C** (보통) | 5-14 | 100-1,000 | 5-14 | 제한적 |
| **D** (초기) | <5 | <100 | <5 | 없음 |
