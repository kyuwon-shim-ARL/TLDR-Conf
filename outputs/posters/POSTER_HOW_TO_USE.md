## 💡 실전 팁

### 1. 사전 준비 (학회 2-3일 전)

```bash
✅ 통합 인덱스에서 키워드 검색
✅ 관심 포스터 10-15개 선정
✅ 코드 리스트 출력 또는 스마트폰에 메모
✅ Zone 배치 확인
✅ 발표자에게 물어볼 질문 준비
```

### 2. 학회 당일

**시간 관리**:
- 포스터 1개당 3-5분 (빠르게)
- 정말 관심 있는 것은 10분
- 90분 세션에 최대 15-18개 가능
- 하지만 현실적으로 10-12개가 적당

**명함 준비**:
- 최소 20장 지참
- 관심 있는 포스터마다 교환
- 뒷면에 포스터 코드 메모

**질문 예시**:
- "What was the most interesting finding?"
- "Are you planning to publish this?"
- "Which method did you use for [analysis]?"
- "We're working on similar topics. Can we collaborate?"

### 3. 포스터 에티켓

**DO**:
✅ 발표자가 설명 중이면 끝날 때까지 기다리기
✅ 자기소개 후 질문
✅ 사진 촬영 시 허락 받기
✅ 명함 교환

**DON'T**:
❌ 설명 중 끼어들기
❌ 너무 오래 독점 (뒤에 대기자 배려)
❌ 무단 촬영
