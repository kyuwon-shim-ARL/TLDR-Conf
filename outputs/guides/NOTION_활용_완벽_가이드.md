# 2. 학회 자료 임포트하기

## 2.1 마크다운 파일 임포트 (가장 쉬운 방법)

### 방법 1: 드래그 앤 드롭

```
1. 노션 페이지 열기
2. 마크다운 파일(.md) 드래그해서 노션 페이지로 끌어놓기
3. "Import" 선택
4. 자동 변환 완료!
```

**임포트할 파일**:
- `NOTION_all_sessions_guide.md`
- `LAB_SHARING_comprehensive_guide.md`
- `background_S1.md` ~ `background_S21.md`

### 방법 2: Import 메뉴 사용

```
1. 노션 좌측 사이드바에서 "Import" 클릭
2. "Markdown" 선택
3. 파일 선택 (여러 파일 동시 선택 가능)
4. "Import" 클릭
```

---

## 2.2 구조화된 임포트 전략

### 추천 폴더 구조

```
📁 MSK2025 학회 (Root 페이지)
├─ 📄 학회 개요
├─ 📁 세션별 배경 자료
│  ├─ 📄 S1: Bacterial Pathogenesis
│  ├─ 📄 S2: Diet-Microbiome-Host
│  ├─ ...
│  └─ 📄 S21: Microbial Biotechnology
├─ 📁 개인별 참석 계획
│  ├─ 📄 박사후연구원 A (AMR 전문)
│  ├─ 📄 박사과정 B (Microbiome 전문)
│  └─ 📄 석사과정 C
├─ 📁 공유 자료
│  ├─ 📄 전체 일정표
│  ├─ 📄 네트워킹 전략
│  └─ 📄 숙박/교통 정보
└─ 📁 학회 후 정리
   ├─ 📄 주요 발견 (Key Findings)
   ├─ 📄 명함 정리
   └─ 📄 Follow-up 계획
```

### 단계별 설정

```
Step 1: Root 페이지 만들기
1. 노션 좌측 사이드바 "Add a page"
2. 제목: "🧬 MSK2025 학회"
3. Icon 추가 (🧬, 🦠, 🔬 등)
4. Cover 추가 (선택사항)

Step 2: 하위 페이지 만들기
1. Root 페이지 안에서 "/page" 입력
2. "Page" 선택
3. 제목 입력 후 Enter
4. 또는 "+" 버튼 클릭 → "Page" 선택

Step 3: 마크다운 파일 임포트
1. 각 하위 페이지로 드래그 앤 드롭
2. 또는 페이지 내에서 "/import" 입력
```

# 3. 페이지 구조화하기

## 3.1 효과적인 노션 블록 활용

### Toggle 블록 (접기/펼치기)

**언제 사용**: 긴 내용을 숨기고 필요할 때만 펼치기

**만드는 법**:
```
1. "/toggle" 입력
2. 제목 입력 (예: "발표 1: Wonsik Lee")
3. 내용 작성
```

**예시**:
```
> 발표 1: Wonsik Lee (SKKU)
  - S. aureus niche adaptation
  - Tn-Seq으로 host niche-specific genes 분석
  - 질문: "Tn-Seq를 도심 환경에 적용 가능한가요?"
```

---

### Callout 블록 (중요 정보 강조)

**만드는 법**:
```
1. "/callout" 입력
2. 아이콘 선택 (💡, ⚠️, ✅ 등)
3. 배경색 선택
```

**예시**:
```
💡 최우선 네트워킹
Eun-Kyung Lim (KRIBB) - CN-TAR 공기 중 AMR 검출
발표 직후 질문 필수!
```

---

### 테이블 (비교 정보)

**만드는 법**:
```
1. "/table" 입력
2. 열 추가/삭제
3. 정렬, 필터 가능
```

**예시 - 세션 선택 테이블**:

| 세션 | 점수 | 추천 대상 | 시간 |
|------|------|-----------|------|
| S4 | 95 | AMR 연구자 | 10/27 09:00 |
| S8 | 98 | 진단 기술 | 10/27 13:50 |

---

### 체크박스 (To-do)

**만드는 법**:
```
1. "[]" 입력 후 Space
2. 또는 "/todo" 입력
```

**예시**:
```
학회 전 준비
- [ ] 워크샵 등록 (W1, W5)
- [ ] 명함 준비 (한글/영문 각 20장)
- [ ] S4 연사 논문 읽기
- [ ] 질문 리스트 작성
```

---

### 데이터베이스 (고급 활용)

**언제 사용**: 여러 세션을 필터링/정렬하고 싶을 때

**만드는 법**:
```
1. "/database" 입력
2. "Table" 선택
3. 속성(Property) 추가
   - 세션명 (Title)
   - 날짜 (Date)
   - 점수 (Number)
   - 분야 (Select)
   - 참석 여부 (Checkbox)
```

**예시 구조**:

| 세션명 | 날짜 | 점수 | 분야 | 참석 | 메모 |
|--------|------|------|------|------|------|
| S4 호흡기 병원체 | 10/27 | 95 | AMR | ✅ | Jinki Yeom 발표 주목 |
| S8 진단 | 10/27 | 98 | 진단 | ✅ | CN-TAR 최우선 |

**필터 활용**:
```
- "분야" = AMR → AMR 관련 세션만 보기
- "점수" > 80 → 고득점 세션만 보기
- "참석" = ✅ → 참석 예정 세션만 보기
```

# 4. 랩 구성원과 공유하기

## 4.1 워크스페이스 초대

### 팀원 초대 방법

```
1. 좌측 사이드바 "Settings & members" 클릭
2. "Members" 탭
3. "Invite" 클릭
4. 이메일 입력 (여러 명 동시 가능)
5. 권한 선택:
   - Full member (편집 가능) ← 추천
   - Guest (특정 페이지만 접근)
6. "Invite" 클릭
```

**💡 Tip**: 대학 이메일로 초대하면 Education 플랜 무료 사용

---

## 4.2 페이지 공유 링크

### 외부 공유 (읽기 전용)

```
1. 공유할 페이지 우측 상단 "Share" 클릭
2. "Share to web" 토글 ON
3. "Copy link" → 링크 복사
4. 카톡/이메일로 전송
```

**주의**: 누구나 링크만 있으면 접근 가능 (비밀번호 없음)

---

### 팀 내부 공유 (편집 가능)

```
1. "Share" 클릭
2. "Invite" 탭에서 팀원 선택
3. 권한 선택:
   - Can edit (편집 가능) ← 추천
   - Can comment (댓글만)
   - Can view (읽기 전용)
4. "Invite" 클릭
```

---

## 4.3 권한 관리 전략

### 추천 권한 구조

| 역할 | 권한 | 이유 |
|------|------|------|
| **PI** | Full access (Owner) | 전체 관리 |
| **박사후연구원** | Can edit | 자료 추가/수정 |
| **대학원생** | Can edit | 개인 계획 편집 |
| **학부 인턴** | Can view | 읽기만 |

**💡 Tip**: 중요 페이지는 "Lock page" (우측 상단 ⋯ → Lock page)로 실수 편집 방지

# 5. 협업 기능 활용하기

## 5.1 댓글 (Comments)

### 사용법

```
1. 텍스트 드래그
2. "Comment" 클릭 (말풍선 아이콘)
3. 댓글 작성
4. "@" 입력 후 팀원 태그 → 알림 전송
```

**예시 시나리오**:
```
연구원 A: "S8 세션에서 CN-TAR 발표 듣고 싶은데,
          누가 같이 질문할래요?"
          @박사과정B @석사과정C

박사과정 B: "@연구원A 저요! 질문 리스트 공유해주세요."
```

---

## 5.2 실시간 협업

### 동시 편집

- 여러 명이 동시에 같은 페이지 편집 가능
- 우측 상단에 접속 중인 사람 프로필 표시
- 커서 위치 실시간 표시 (색상으로 구분)

### 버전 히스토리 (Page history)

```
1. 우측 상단 "⋯" → "Page history"
2. 과거 버전 확인
3. "Restore this version" 클릭 → 이전 버전 복구
```

**💡 Tip**: Education 플랜은 30일 버전 히스토리 제공

# 8. FAQ & 트러블슈팅

## 8.1 자주 묻는 질문

### Q1. 노션 무료 플랜으로 충분한가요?

**A**: Education 플랜(무료)이면 충분합니다!
- 대학 이메일로 가입 → Education 플랜 자동 제공
- Plus 기능 모두 무료

---

### Q2. 마크다운 파일 임포트 시 서식이 깨져요.

**A**: 다음 확인:
1. 파일 인코딩이 UTF-8인지 확인
2. 이미지 링크가 로컬 경로가 아닌지 확인 (상대 경로 권장)
3. 테이블 양옆에 빈 줄 추가

**해결법**:
```
수동 복사-붙여넣기:
1. 마크다운 파일을 텍스트 에디터로 열기
2. 전체 복사 (Ctrl+A → Ctrl+C)
3. 노션 페이지에 붙여넣기 (Ctrl+V)
4. 노션이 자동으로 변환
```

---

### Q3. 오프라인 모드가 작동하지 않아요.

**A**:
1. 앱 버전 최신인지 확인
2. Settings → Offline mode → Enable 확인
3. 페이지를 사전에 열어두기 (캐시 생성)
4. Wi-Fi 끄고 테스트

---

### Q4. 모바일에서 편집이 불편해요.

**A**:
- **단축키 활용**:
  - "/" → 블록 메뉴
  - "@" → 사람 태그
  - "[]" → 체크박스
- **음성 입력** 사용 (키보드 마이크 버튼)
- **Quick capture**: 노션 위젯으로 빠른 메모

---

### Q5. 여러 명이 동시에 편집하면 충돌 나나요?

**A**:
- 노션은 실시간 동기화로 충돌 없음
- 같은 블록을 동시 편집 시 마지막 저장이 적용됨
- **해결법**: 각자 다른 섹션 편집 or 댓글로 협업

---

### Q6. 페이지가 너무 길어서 로딩이 느려요.

**A**:
1. **Toggle 블록 활용** → 대부분 접어두기
2. **Subpage로 분리** → 긴 내용은 별도 페이지
3. **Linked database** 사용 → 데이터베이스는 참조만

---

### Q7. 공유 링크를 취소하고 싶어요.

**A**:
```
1. "Share" 클릭
2. "Share to web" 토글 OFF
3. 기존 링크 무효화됨
```

---

## 8.2 트러블슈팅

### 문제: 페이지가 삭제되었어요!

**해결**:
```
1. 좌측 사이드바 "Trash" 클릭
2. 삭제된 페이지 찾기
3. "Restore" 클릭 → 복구
```

**주의**: 30일 후 영구 삭제

---

### 문제: 동기화가 안 돼요.

**해결**:
```
1. 인터넷 연결 확인
2. 앱 재시작
3. 로그아웃 → 로그인
4. 페이지 새로고침 (Ctrl+R)
```

---

### 문제: 이미지가 보이지 않아요.

**해결**:
```
1. 이미지 URL이 유효한지 확인
2. 로컬 이미지는 노션에 업로드 (드래그 앤 드롭)
3. 파일 크기 확인 (Free 플랜: 5MB 제한)
```

---

### 문제: 테이블이 모바일에서 안 보여요.

**해결**:
```
1. 테이블 가로 스크롤 사용
2. 또는 "Simple table" 사용 (작은 테이블)
3. 모바일용으로 List 뷰 추가 (데이터베이스)
```

## Tip 2: 템플릿 활용

**노션 템플릿 갤러리**:
```
1. 좌측 사이드바 "Templates"
2. "Conference" 또는 "Meeting notes" 검색
3. "Use this template" → 자동 복사
```

**커스텀 템플릿 저장**:
```
1. 자주 쓰는 페이지 구조 만들기
2. 우측 상단 "⋯" → "Duplicate"
3. 필요할 때마다 복제
```

---

## Tip 3: 북마크 활용

**외부 링크 임베드**:
```
1. "/bookmark" 입력
2. URL 입력 (학회 홈페이지, 논문 등)
3. 프리뷰 자동 생성
```

**예시**:
```
/bookmark https://msk.or.kr/conference
→ 학회 홈페이지 프리뷰 카드 생성
```

---

## Tip 4: Synced Blocks (동기화 블록)

**같은 내용을 여러 페이지에 표시**:
```
1. 원본 블록 드래그
2. "⋮⋮" 클릭 → "Turn into synced block"
3. 다른 페이지에서 "/synced" → 원본 선택
4. 한 곳만 수정하면 모든 곳이 업데이트
```

**활용 예시**:
- 일정표를 여러 페이지에 동기화
- 네트워킹 타겟을 개인 계획 & 공유 페이지에 표시

---

## Tip 5: Web Clipper (크롬 확장)

**논문/웹페이지를 노션으로 저장**:
```
1. Chrome Web Store에서 "Notion Web Clipper" 설치
2. 학회 홈페이지, 연사 논문 페이지에서 클릭
3. 저장할 노션 페이지 선택
4. 전체 페이지가 노션에 저장됨
```
